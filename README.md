Projeto de Finalização de Módulo IV

No módulo atual do meu curso Técnico em T.I., realizamos diversas atividades — e entre elas, a principal e mais desafiadora foi o desenvolvimento de uma loja virtual de e-commerce totalmente funcional. O projeto incluiu carrinho de compras, conversor de moedas (quando necessário), catálogo de produtos e diversas outras funcionalidades desenvolvidas de forma original.

Para construir o site, utilizamos tecnologias como HTML, CSS, JavaScript e também Photoshop para criação e edição de imagens. Além disso, buscamos referências em outros sites até chegarmos ao resultado final. Nossa principal fonte de inspiração foi o Caldeirão Místico, um e-commerce real especializado em artigos esotéricos.
